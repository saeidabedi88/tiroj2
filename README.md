# TirojNet Website

Modern website for TirojNet, built with Next.js, React, and Tailwind CSS.

## Features

- Modern, responsive design
- Server-side rendering with Next.js
- Styled with Tailwind CSS
- Interactive UI components
- SEO optimized

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/saeidabedi88/tiroj2.git
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Build for production:
```bash
npm run build
# or
yarn build
```

5. Start production server:
```bash
npm run start
# or
yarn start
```

## Deployment

This project can be deployed to any hosting platform that supports Node.js. Follow these steps for deployment:

1. Build the project:
```bash
npm run build
```

2. Start the production server:
```bash
npm run start
```

The site will be available at `http://localhost:3000`

## Environment Variables

Create a `.env` file in the root directory with the following variables:

```env
NEXT_PUBLIC_SITE_URL=https://tirojnet.ca
```
